Este proyecto se enfocó en la utilización de la arquitectura CSS BEM (Block, Element, Modifier). BEM es una metodología que se centra en la creación de componentes reutilizables. La idea es que todo en una página web es un bloque y cada bloque puede contener elementos y modificadores. Además, se utilizó HTML y CSS para desarrollar el proyecto
